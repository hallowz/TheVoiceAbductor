# Voice Abductor

[![Watch a demo](https://img.youtube.com/vi/m9_sDgwtPkY/maxresdefault.jpg)](https://www.youtube.com/watch?v=m9_sDgwtPkY)

## You'll get the Voice Abuctor - a program that lets you:

- Record your team or friends in **Valorant, Discord, or any VOIP**
- Instantly clip the **last 20 seconds of audio**
- Automatically detect phrases and **instantly capture and playback within a second**
- Assign sounds to **hotkeys for instant playback**
- Trim & clean clips with a **built-in editor**
- Uses **auto speech-to-text** with the ability to edit the transcripts
- Save, organize, and spam voice lines like a soundboard
